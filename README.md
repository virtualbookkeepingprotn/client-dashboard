# Mako Shade Financial Dashboard

A password-protected financial dashboard for Mako Shade.

## Access Codes
- **mako2025** - Owner access
- **accountant123** - Accountant access

## Deployment to Vercel via GitHub

### Step 1: Create GitHub Repository
1. Go to github.com and sign in
2. Click the **+** icon (top right) â†’ **New repository**
3. Name it: `mako-shade-dashboard`
4. Keep it **Private**
5. Click **Create repository**

### Step 2: Upload Files
1. On your new repository page, click **uploading an existing file**
2. Drag and drop ALL files from this folder
3. Click **Commit changes**

### Step 3: Deploy to Vercel
1. Go to vercel.com and sign in with GitHub
2. Click **Add New Project**
3. Find `mako-shade-dashboard` and click **Import**
4. Click **Deploy**
5. Wait 1-2 minutes for deployment

### Step 4: Get Your Link
Once deployed, Vercel gives you a URL like:
`https://mako-shade-dashboard.vercel.app`

Share this link + the access code with your client!

## Updating Data
To update the financial data, edit the `pages/index.js` file and change the numbers in the data arrays.
